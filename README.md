# Azure KMU VDI Lab

Production-like Azure infrastructure and business operations lab for a fictional 50-user industrial company.

## Goals

- Azure VDI infrastructure
- Zero Trust network segmentation
- CMDB and asset management
- 50 fictional employees and user accounts
- RBAC and Entra ID group model
- ITSM ticket system
- Security baseline
- Backup and recovery model
- Monitoring and SIEM model
- FastAPI control API
